### 2. Setup
* Add jitpack dependency to root build.gradle
 ```bash
        	allprojects {
        		repositories {
        			...
        			maven { url 'https://jitpack.io' }
        		}
        	}
 ```
 
 * Add the dependency 
  ```bash
	dependencies {
	        implementation 'com.github.UsePace:android-message-center:1.0'
	}
  ```
